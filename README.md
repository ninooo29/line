##association

###User
- has_many :users_groups
- has_many :groups, through: :users_groups
- has_many :messages

###Message
- belongs_to :user

###Group
- has_many :users_groups
- has_many :users, through: :users_groups
- has_many :messages

###User_Group
- belongs_to :user
- belongs_to :group
